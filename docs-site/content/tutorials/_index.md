Start here for step-by-step guides to install, deploy, and operate Symphony.

* [Run Symphony as a Single Process]({{< relref "/tutorials/run-symphony-as-single-process" >}})

	Run Symphony entirely in memory for lightweight tests and experiments. When you are done, just exit the process, with no cleanup and no leftovers.

* [Deploy Symphony to Kubernetes]({{< relref "/tutorials/run-symphony-on-k8s" >}})

	Deploy Symphony to Kubernetes for production environments. Interact with Symhony through either REST API or native Kubernetes tools like `kubectl`.
